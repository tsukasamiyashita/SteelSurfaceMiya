SteelSurfaceMiya-v1.0.0

アプリの概要

SteelSurfaceMiya は、一般的な鋼材の断面サーフェスを簡単に生成し、CAD形式のファイルとして出力できるデスクトップアプリケーションです。

機能

対応鋼材: H形鋼、I形鋼、T形鋼、角形鋼管、円形鋼管、等辺山形鋼、不等辺山形鋼、溝形鋼、平鋼、角鋼、丸鋼

寸法設定: プリセットからのワンクリック選択、および詳細寸法の手入力によるカスタマイズが可能。
※設定可能な寸法：幅、高さ、厚み、内R（フィレット半径）、先端R（つま先半径）、フランジ勾配（％）

対応出力形式:

3Dサーフェスモデルとして出力: .stl, .obj, .ply, .gltf, .glb, .off

2D断面図（線画）として出力: .dxf

動作環境

Windows 10 / Windows 11 (64bit)

インストール・実行方法

アプリケーションの実行ファイル（.exe）をダウンロードした場合、以下の手順で起動してください。

ダウンロードした SteelSurfaceMiya-v1.0.0.exe を任意のフォルダに保存します。

ファイルをダブルクリックして起動します。

【重要】Windows Defender の警告が表示された場合
初回起動時、「Windows によって PC が保護されました」という青い警告画面（SmartScreen）が表示される場合があります。これはインターネットからダウンロードした新しいアプリに対して表示される一般的なセキュリティ機能です。
以下の手順で安全に起動できます：

警告画面のテキスト内の 「詳細情報」 をクリックします。

右下に 「実行」 ボタンが現れるので、それをクリックするとアプリが起動します。

基本的な使い方

鋼材の選択: 画面左側のパネルで、出力したい鋼材の種類とプリセットサイズを選択します。

寸法設定: 画面中央のパネルで、各寸法を必要に応じて書き換えます。単位はすべてミリメートル（mm）です。
・I形鋼や溝形鋼のフランジ内面にある傾斜は「フランジ勾配(%)」として設定・再現可能です。
・角の丸みは「内R（フィレット）」および「先端R（つま先）」として設定可能です。
※選択した鋼材によって不要な入力欄は自動的に無効化されます。

出力設定: 画面右側のパネルで、出力したいファイル形式（STL, OBJ, PLY, GLTF, GLB, OFF, DXF）を選びます。

保存: 「ファイルを作成して保存」ボタンを押し、名前を付けてファイルを保存してください。保存したファイルはお手持ちの3DCADやビューワーで開くことができます。
